#  Canberk Sezgin - Personal Portfolio Website

This project is a modern, corporate, and fully responsive portfolio website developed to showcase my computer engineering work, projects, and professional background.

Live demo: [canberksezginportfolio.netlify.app](https://canberksezginportfolio.netlify.app)

---

##  Key Features

-  **Multi-Language Support:** Dynamic switching between English and Turkish (Instant toggle without page reload).
-  **Modern & Corporate Design:** Featuring glassmorphism effects, a sleek dark theme, and smooth scroll animations.
-  **Fully Responsive:** Optimized interface for desktop, tablet, and mobile devices.
-  **Performance Oriented:** High-performance execution using Vanilla JavaScript and modern CSS3 with zero external library dependencies.
-  **Integrations:** Direct LinkedIn link, email integration, and a downloadable CV button.

---

##  Technology Stack

- **HTML5:** Semantic structure and SEO optimization.
- **CSS3:** Custom Variables (CSS Variables), Flexbox, Grid, and Keyframe animations.
- **JavaScript (ES6+):** Dynamic content management and translation system.
- **Hosting:** Netlify (Integrated with GitHub for CI/CD).

---

##  Project Highlights

Key projects featured in the portfolio:
*   **Exys:** A full-stack Auto Expertise Management System built with Django.
*   **MoodFood:** An AI-powered (Ollama3/GPT) mobile application for meal suggestions based on user mood.
*   **Image Classification:** An Explainable AI (XAI) deep learning pipeline using PyTorch and VGG19.
